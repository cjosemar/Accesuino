Ayuda gestión Accesduino

Las empresas tienen la obligación de llevar a cabo un registro diario de las horas que realice cada empleado, a fin de poder asegurar el control de las horas extraordinarias. Este control, según la Audiencia Nacional (AN), debe realizarse incluso en aquellas compañías en las que no se hagan horas extra.

El Real Decreto-ley 8/2019, de 8 de marzo, de medidas urgentes de protección social y de lucha contra la precariedad laboral en la jornada de trabajo, determina que "el registro de jornada, que no de horas extraordinarias, es el requisito constitutivo para controlar los excesos de jornada". Además, asevera que la negación de este registro "coloca a los trabajadores en situación de indefensión que no puede atemperarse porque las horas extraordinarias sean voluntarias, puesto que el único medio de acreditarlas es, precisamente, el control diario".

La creación de un registro, concluye la resolución, eliminará cualquier duda sobre si se hacen o no horas extraordinarias y si su realización es voluntaria.

Con Accesduino quedara resuelto todos estos problemas y se realiza con la comodidad de poder realizar los registros con la ultima tecnología. Para ellos utilizaremos Tarjeta RFID que identificara a cada empleado y recogerá el sistema tanto la hora de entrada como la de salida y el calculo de las horas de trabajo, extra y absentismo de cada empleado.

Gestión de Usuarios

Dentro del menú Administración - Usuarios - Gestión de usuarios aparecerá la siguiente pantalla donde se pueden crear, modificar o eliminar los sucesivos usuarios y mostrando los datos de cada usuario una vez que se selecciona.

Gestión de Roles

Dentro del menú Administración - Usuarios - Roles aparecerá la siguiente pantalla donde se pueden crear, modificar o eliminar los sucesivos roles y mostrando los datos de cada role una vez que se selecciona.

Datos Empresa

Dentro del menú Administración - Datos de empresa aparecerá la siguiente pantalla donde se pueden modificar los datos de la empresa.

Gestión de Empleados

Dentro del menú Empleados - Empleados aparecerá la siguiente pantalla donde se pueden crear, modificar o eliminar los sucesivos Empleados y mostrando los datos de cada Empleado una vez que se selecciona.

Gestión de Puestos

Dentro del menú Empleados - Puestos aparecerá la siguiente pantalla donde se pueden crear, modificar o eliminar los sucesivos Puestos y mostrando los datos de cada Puesto una vez que se selecciona.

Gestión de Horarios

Dentro del menú Empleados - Horarios aparecerá la siguiente pantalla donde se pueden crear, modificar o eliminar los sucesivos Horarios y mostrando los datos de cada Horario una vez que se selecciona.

En la creación o modificación del horario hay que introducir el inicio y el fin o la duración y el tercer dato se calcula automáticamente.

Gestión de Tipos de Permisos

Dentro del menú Permisos - Tipos Permiso aparecerá la siguiente pantalla donde se pueden crear, modificar o eliminar los sucesivos Tipos de Permiso y mostrando los datos de cada Tipo de Permiso una vez que se selecciona.

Gestión de Registros

Dentro del menú Registros - Administrar Registros aparecerá la siguiente pantalla donde se pueden modificar o eliminar los sucesivos Registros y mostrando los datos de cada Registro una vez que se selecciona.

En la columna Estado queda reflejado todos los registros que no estan completos. Por lo tanto refleja que están en su puesto de trabajo.

El total de horas se calcula automático redondeando la horas transcurridas entre Inicio y Fin.
